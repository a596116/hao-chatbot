# 更新日誌

所有重要的變更都將記錄在此文件中。

本項目遵循[語義化版本](https://semver.org/lang/zh-TW/)規範。

## [1.0.0] - 2024-11-20

### 新增
- ✨ 初始版本發布
- 🤖 浮動聊天機器人組件
- 🎨 美觀的 UI 設計，支持動畫效果
- 📱 響應式佈局，完美支持移動端
- 🔧 可自定義位置、大小、標題等配置
- 📦 支持 npm 和 CDN 引入
- 💬 支持用戶消息和 AI 回覆
- ⌨️ 支持 Enter 發送、Shift+Enter 換行
- 📝 TypeScript 支持
- 🎯 事件系統（message-sent、message-received）
- 📚 完整的文檔和使用示例

### 技術棧
- Vue 3.4+
- TypeScript 5.3+
- Vite 5.1+
- Vue Element Plus X 1.2+

## 版本格式說明

### Added（新增）
新增的功能。

### Changed（變更）
現有功能的變更。

### Deprecated（棄用）
即將移除的功能。

### Removed（移除）
已移除的功能。

### Fixed（修復）
任何 bug 修復。

### Security（安全性）
安全性相關的修復或改進。
